# Sentiment Analysis

Sentiment analysis neural network trained by fine tuning BERT on the Stanford Sentiment Treebank.

## Train network
```
python3 train.py
```

## Evaluate network
```
python3 evaluate.py
```

## Analyze sentiments of your own inputs
```
python3 analyze.py
```

## Run server
```
python3 server.py
```

## Setup client
```
cd client
npm install
```

## Run client
```
cd client
npm run serve
```
